# 术语解释

## 表级术语

### 論理テーブル名

用于表达业务语义，主要服务阅读与沟通。

### 物理テーブル名

数据库真实建表名，会直接进入 DDL。

## 字段级术语

### 論理名

字段语义名称，强调业务可读性。

### 物理名

字段真实列名，会直接成为 SQL 列标识符。

### データ型

字段类型定义，如 `varchar`、`int`、`date`。

### Size

字段长度或精度信息，如 `255` 或 `10,2`。

## 约束术语

### PK

主键标记，系统据此生成主键约束。

### Not Null

字段不能为空，DDL 中表现为 `NOT NULL`。

### 備考

补充说明字段，通常用于注释语义。
