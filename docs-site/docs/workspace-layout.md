# 工作台总览

工作台是整条链路的控制面板。你在这里完成输入、定位、预览和生成。

![工作台总览](/img/screens/app-main-overview-hd.png)

## 页面结构

- 文件区：上传、切换、删除文件
- Sheet 区：切换当前解析范围
- 预览区：确认表与字段是否被正确识别
- 生成区：设置方言、导出模式和修复动作
- 顶部状态：显示当前文件与当前 Sheet

## 常用动作

文件区底部提供三个长期入口：

- 文档：跳转在线手册
- 设置：进入全局配置页
- 语言：切换界面语言

预览区支持结构视图和电子表格视图。前者用于确认解析结果，后者用于缩小解析范围。

生成区聚合最终交付动作，包括 SQL 生成、复制、导出和 Name Fix。

## 完成信号

当下面三个信号同时出现，说明工作台状态稳定：

- 已有可用文件和可切换 Sheet
- 预览区出现完整表结构
- 生成区动作按钮可用

进入下一步可参考 [第一次导入 Excel](./first-import.md) 与 [从预览到 DDL 生成](./preview-to-ddl.md)。
